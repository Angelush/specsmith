# The One AI Writing Hack Nobody Talks About.

**Video:** [https://www.youtube.com/watch?v=ltbzgzZZmgI](https://www.youtube.com/watch?v=ltbzgzZZmgI) • **ID:** `ltbzgzZZmgI` • **Playlists:** P1, P2

## Concepts covered

- [[concepts/ai-productivity-workflows]] — The "project room / data room": a bounded local workspace for one serious job; the first instruction is never "do the thing" but "find the materials and build the room" — modern agents (Opus 4.7, GPT-5.5) walk folder trees, compare dates, and inspect metadata.
- [[concepts/advanced-prompting-techniques]] — You cannot prompt away hallucination (no truth-check pass inside the model); the structural fix is four artifacts — source inventory, conflict log, missing-context list, duplicates report — after which the writing prompt gets short.
- [[concepts/ai-quality-control]] — Organizational/structural hallucination (Sullivan & Cromwell's fabricated citations) comes from the messy environment around the model; the source-inventory table makes the agent's judgment legible and is a clean review gate.
