## Concepts covered

- [[concepts/agentic-commerce]] — An agentic purchase unbundles into six contested layers (shop / authorization / credential / rails / governance / responsibility); ACP (OpenAI+Stripe checkout) vs UCP (Shopify+Google merchant control), and authorization (Google AP2 mandates) is not the same as payment.
- [[concepts/ai-infrastructure-evolution]] — Machine-to-machine payments favor stablecoin rails (USDC) via Coinbase x402 (reviving HTTP 402) and Stripe's machine payments protocol for tiny, frequent, software-native payments, while cards/wallets persist for consumer protection.
- [[concepts/ai-industry-competitive-landscape]] — The platform owning the agent runtime (AWS Bedrock AgentCore Payments) wins governance leverage over the payment providers; platform commerce stays controlled (Amazon resists unauthorized agentic browsing), not a march to open interoperability.
