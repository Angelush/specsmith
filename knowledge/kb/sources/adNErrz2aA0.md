# Your SaaS Bill Just Got a Second Meter. You're About to Pay It.

**Video:** [https://www.youtube.com/watch?v=adNErrz2aA0](https://www.youtube.com/watch?v=adNErrz2aA0) • **ID:** `adNErrz2aA0` • **Playlists:** P0

## Concepts covered

- [[concepts/ai-economy-and-bottlenecks]] — SaaS is adding a "second meter" for delegated agent work on top of per-seat pricing (Salesforce Agentforce billing "agentic work units," Microsoft Copilot credits, ServiceNow action fabric); the seat was always a proxy for human value, now metered as delegated work.
- [[concepts/ai-business-strategy]] — "Pricing follows platform control": the vendor that defines the work primitive earns the right to price the work, so building on a platform without understanding its metering incentives hands your agent economics to someone else.
- [[concepts/enterprise-ai-adoption]] — Fair vs rent-seeking agent licenses (visible meter, no billing failed work, governed third-party path, read/write/approve/execute rates, caps); SAP's 2026 API policy makes outside-agent access a contractual question; negotiate agent access before usage is mission-critical.
- [[concepts/agent-orchestration-architecture]] — Cost-aware agent design needs an operation taxonomy (read vs write vs approve vs execute, reversibility, blast radius); an agent that treats every tool call the same is "an incident waiting to happen."
